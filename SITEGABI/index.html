if-ELSE

A estrutura if-else é condicional e é usada para tomar decisões com base em uma condição estabelecida.
Neste exemplo abaixo, o programa solicita a idade do usuário e, com base na idade fornecida, decide se o usuário é maior ou menor de idade.: 

#include <stdio.h>
          
int main()
{
    int idade; // Declaração de uma variável 'idade' para armazenar a idade
                  do usuário.

    printf("Digite sua idade: "); // Exibe uma mensagem solicitando que o
                                     usuário digite sua idade
    scanf("%d", &idade); // Lê a idade fornecida pelo usuário e a armazena
                            na variável 'idade'

    if (idade >= 18) // Início de uma estrutura condicional 'if', verifica
                        se a idade é maior ou igual a 18.
    {
        printf("Você é maior de idade.\n"); // Se a condição for verdadeira,
                                               exibe a mensagem de que o
                                               usuário é maior de idade.
    }
    else // Caso a condição do 'if' seja falsa
    {
        printf("Você é menor de idade.\n"); // Exibe a mensagem de que o 
                                               usuário é menor de idade.
    }

    return 0; // Retorna 0 para indicar que o programa foi concluído com
                 sucesso.
}

Switch-CASE

A estrutura switch-case é usada para selecionar uma ação com base no valor de uma expressão.
Neste exemplo abaixo, o programa permite que o usuário escolha uma opção e, com base na escolha, executa o código associado a essa opção. 

#include 

int main()
{
    char opcao;  // Declaração de uma variável 'opcao' para armazenar
                   a escolha do usuário

    printf("Escolha uma opção (A, B, C, ou D): "); // Exibe uma 
    mensagem solicitando que o usuário escolha uma opção
    scanf(" %c", &opcao); // Lê a escolha do usuário (um 
    caractere) e a armazena na variável 'opcao'

    switch (opcao) // Início de uma estrutura de seleção 'switch'
                      com base no valor da variável 'opcao'
    {
    case 'A': // Se a 'opcao' for igual a 'A', execute o seguinte
                 bloco de código:
        printf("Você escolheu a opção A.\n");  // Exibe a mensagem
      indicando que o usuário escolheu a opção 'A'
        break; // Sai da estrutura 'switch'
    case 'B':// Se a 'opcao' for igual a 'B', execute o seguinte
      bloco de código:
        printf("Você escolheu a opção B.\n"); // Exibe a mensagem
      indicando que o usuário escolheu a opção 'B'
        break; // Sai da estrutura 'switch'
    case 'C': // Se a 'opcao' for igual a 'C', execute o seguinte
     bloco de código:
        printf("Você escolheu a opção C.\n"); // Exibe a mensagem
      indicando que o usuário escolheu a opção 'C'
        break; // Sai da estrutura 'switch'
    case 'D': // Se a 'opcao' for igual a 'D', execute o seguinte
     bloco de código:
        printf("Você escolheu a opção D.\n"); // Exibe a mensagem
      indicando que o usuário escolheu a opção 'D'
        break; // Sai da estrutura 'switch'

    default: // Se a 'opcao' não corresponder a nenhuma das opções
      anteriores, execute o seguinte bloco de código:
        printf("Opção inválida.\n"); // Exibe a mensagem indicando
      que a opção é inválida
    }

    return 0; // Retorna 0 para indicar que o programa foi concluído
    com sucesso.
}

while

A estrutura de repetição while é usada para executar um bloco de código repetidamente enquanto uma condição é verdadeira.
Neste exemplo abaixo, o programa faz uma contagem de 1 a 5 usando um loop while. O bloco de código dentro do while é executado enquanto a condição contador <= 5 for verdadeira. 

  #include 

    int main()
    {
        int contador = 1; // Declaração de uma variável 'contador' e
         inicialização com o valor 1
    
        while (contador <= 5) // Início de um loop 'while', que continuará
        enquanto a condição (contador <= 5) for verdadeira
        {
            printf("Contagem: %d\n", contador); // Exibe a mensagem com o
            valor atual de 'contador'
            contador++; // Incrementa o valor de 'contador' em 1 unidade
        }
    
        return 0; // Retorna 0 para indicar que o programa foi concluído
        com sucesso.
    }

for

A estrutura de repetição "for" é usada para executar um bloco de código dde um número específico de vezes.
Neste exemplo abaixo, o programa realiza a mesma contagem de 1 a 5, mas utiliza um loop for para controlar a iteração. A estrutura for é composta por três partes: a inicialização (int i = 1), a condição de continuação (i <= 5), e a expressão de incremento (i++). O bloco de código dentro do for é executado repetidamente enquanto a condição for verdadeira. 

  #include 
  
    int main()
    {
        for (int i = 1; i <= 5; i++) // Início de um loop 'for' com a
       inicialização (int i = 1), condição (i <= 5) e expressão de incremento (i++)
        {
            printf("Contagem: %d\n", i); // Exibe a mensagem com o valor
            atual de 'i'
        }
    
        return 0; // Retorna 0 para indicar que o programa foi concluído
        com sucesso.
    }
